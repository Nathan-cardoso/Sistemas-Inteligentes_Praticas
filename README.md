# Utilização dos códigos

Para a utilização dos códigos deste repositório, é recomendado o uso de um ambiente virtual do Python (`venv`) para gerenciar as dependências do projeto. Todas as dependências estão nos arquivos **requirements.txt** e **requeriments-linux.txt**. Ambos contém as depedências necessárias, porém, são específicos para cada SO. Logo, usuários Windows podem utilizar o (`requirements.txt`) e usuários linux utilizarem (`requirements-linux.txt`)

Para realizar a instação do Ambiente em ambos os sistemas, segue abaixo o guia de comandos.

### Linux
```bash
# Para a criação do venv utilize
python3 -m venv .venv

# Para entrar no ambiente virtual utilize
source .venv/bin/activate

# Para baixar as dependências necessárias é necessário baixá-las do arquvo requirements.txt
pip install -r requirements-linux.txt

```

### Windows

Ao abrir o CMD utileze os comandos abaixo

```bash
# Para a criação do ambiente 
python -m venv venv

# Para entrar no ambiente virtual
venv\Scripts\activate

# Para baixar as depências necessárias
pip install -r requirements.txt

```
