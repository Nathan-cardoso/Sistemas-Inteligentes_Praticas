# 📚 Repositório - Sistemas Inteligentes (PET1702)

Este repositório é destinado para as práticas da disciplina de **Sistemas Inteligentes (PET1702)**.

Para a utilização dos códigos deste repositório, é recomendado o uso de um ambiente virtual do Python (`venv`) para gerenciar as dependências do projeto.

---

# Configuração do Ambiente Virtual (venv)
Ao abrir o bash utilize os comandos abaixos

---

### Linux
```bash
# Para a criação do venv utilize
python3 -m venv .venv

# Para entrar no ambiente virtual utilize
source .venv/bin/activate

# Para baixar as dependências necessárias é necessário baixá-las do arquvo requirements.txt
pip install -r requirements.txt

```

### Windows

---

Ao abrir o CMD utileze os comandos abaixo

```bash
# Para a criação do ambiente 
python -m venv venv

# Para entrar no ambiente virtual
venv\Scripts\activate

# Para baixar as depências necessárias
pip install -r requirements.txt

```
